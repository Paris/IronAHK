IronAHK is a rewrite of AutoHotkey, a powerful and easy to use scripting language for automation on the desktop
It is developed in C# for .NET and Mono which brings cross-platform compatibility, increased performance, enhanced security, Unicode, native 64-bit support, smaller file sizes for compiled binaries and much more.
