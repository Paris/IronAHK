Developing

Requirements

Windows:

  • Visual C# 2010 Express or greater
  • NUnit 2.4.7
  • msysgit
  • 7-zip
  • WixSharp

Unix:

  • MonoDevelop with NUnit add-in
  • Git

Debian and Ubuntu users should install the following packages from badgerports:

apt-get install monodevelop monodevelop-nunit git-core

Source

The preferred way to obtain the source code is to clone the Git repository which is located at git://gitorious.org/ironahk/mainline.git (http also available).

Contributing

Changes are welcome in the form of patches to the issue tracker. Smaller commits are generally preferred over larger ones. Any new development must pass all NUnit tests.

Building

A release build can be created by running make on all operating systems.

To run in Visual Studio, right click the IronAHK project and select Set as StartUp Project. The file Tests.cs contains more information for running debugging with scripts.
