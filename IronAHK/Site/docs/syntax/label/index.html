Labels

A label identifies a point in a script which can be called or referenced. They are generally discouraged becuase they make scripts less readable and harder to maintain.

Labels take the form of any valid identifier followed by a colon (:) own its own line. Hotkey and hotstring definitions are also labels.

 

goto label

Jumps to the specified label.

 

gosub label

Jumps to the specified label and continues execution until Return is encountered.

 

Examples

gosub MyLabel 
MsgBox, The 'MyLabel' label has returned (it is finished).
return

MyLabel:
MsgBox, The 'MyLabel' label is now running.
return
goto info
MsgBox, This will never be shown.

info:
MsgBox, The 'info' label is now running.
return
