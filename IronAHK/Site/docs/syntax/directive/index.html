Directives

Directive are single line statements which are pre-processed and affect the entire script regardless of where they are placed.
Conditional blocks such as if-statements do not affect directives and with a few exceptions variables may not be used.

#Include[Again] [*i] path

Reads the specified file path and behaves as though the contents are present at this exact position. If path is a directory all future include directives will use this as the parent folder.
The only variables which may be used are A_ScriptDir, A_AppData and A_AppDataCommon.

#Include ensures path is included only once, #IncludeAgain allows multiple inclusions of the same file.

The path parameter may optionally be preceded by *i and a single space, which causes the program to ignore any failure to load the included file.

 

#NoTrayIcon

Disables the showing of the default tray (notification) icon.

 

#SingleInstance [force|ignore|off]

Determines whether a script is allowed to run again when it is already running.

  • force: replaces the old instance automatically, similar to Reload;
  • ignore: leaves the old instance running;
  • off: allow multiple instances to run concurrently.
