MsgBox

The MsgBox command is used to put a simple alert dialog on the screen. For more complicated dialogs and windows, see GUI.

MsgBox text
MsgBox options, title, text

Parameters

options

Indicates the type of message box and the possible button combinations.  If blank or omitted, it defaults to 0.

This parameter will not be recognized if it contains an expression or a variable reference such as %option%. Instead, use a literal numeric value.

title

The title of the message box window. If omitted or blank, it defaults to the name of the script (without path).
text

If all the parameters are omitted, the MsgBox will display the text "Press OK to continue." Otherwise, this parameter is the text displayed inside the message box to instruct the user what to do, or to present information.

Examples

   MsgBox, Hello World
   MsgBox, 16, Critical Error, An unspecified error has occurred
